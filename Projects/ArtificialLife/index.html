How the Random Works

Colony size is the number of simorgs total. The colony is always n x n.

Number of generations is just how many generations you want to simulate.

The initial life rate is the percentage of colony size that starts out alive.

The animation rate is the speed of the animation. (1 Frame / # shown)

Changing the colony size or initial life rate sliders creates a new random colony.

However, changing the # of generations or animation rate does not.

Start the simulation by pressing the "Simulate" button.

How the Simulation Works

Each cell with one or no neighbors dies, as if by loneliness.

Each cell with four or more neighbors dies, as if by overpopulation.

Each cell with two or three neighbors survives.

Each cell with three neighbors becomes populated.

Each cell that ages over 10 generations dies.

Each cell with four or more neighbors dies, as if by overpopulation.
